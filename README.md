# Academic System

A console-based Academic Management System developed in Python for a university assignment demonstrating the use of packages, modules, imports, file handling, validation, and code organization.

## Features

* Add Students
* View Students
* Add Courses
* View Courses
* Assign Grades
* View Grades
* Generate Academic Reports

## Technologies Used

* Python
* Packages and Modules
* Text File Storage (TXT)
* File Handling
* Input Validation

## Project Structure

```text
academic_system/
├── main.py
├── students/
├── courses/
├── grades/
├── reports/
└── data/
```

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:

```bash
python main.py
```

## Assignment Objectives

This project demonstrates:

* Creating and importing modules
* Creating custom packages
* Organizing code into multiple files
* Reading and writing data using text files
* Function reuse across modules
* Building a console-based application

Validation
The system has been tested against the assignment requirements, including:
- Package and module structure
- File handling operations
- Input validation
- Feature completeness

All core requirements have been implemented successfully.

Limitations
- Data is stored in text files (no database integration)
- Console-based interface only
